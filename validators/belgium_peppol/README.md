# Belgium / Peppol Validator Artefacts

Milestone 6B expects official or precompiled validation artefacts to be installed here, but this repository does not commit those artefacts by default.

Expected paths:

- `validators/belgium_peppol/en16931/CEN-EN16931-UBL.xslt`
- `validators/belgium_peppol/peppol/PEPPOL-EN16931-UBL.xslt`

The app keeps EN16931 and Peppol Schematron validation as `not_configured` until an artefact path exists and an XSLT/Schematron engine command is configured.

No production Peppol delivery, recipient acceptance, SMP registration or statutory compliance claim is made by placing files in this folder.
